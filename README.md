# Video Player (XState + React)

## Описание проекта

Тестовое задание на реализацию мини веб-приложения с видеоплеером и управлением состоянием через конечные автоматы.

Функционал:
- Открыть/закрыть видеоплеер
- Поставить на паузу/воспроизвести видео
- Свернуть (мини-режим) и развернуть (полный режим) видеоплеер

Технологии:
- React (Vite)
- XState
- Ant Design
- React Player

## Запуск проекта локально

1. **Клонируйте репозиторий**

```
git clone https://github.com/nathalieMalsh/video-player.git
```

2. **Перейдите в папку проекта**

```
cd frontend-project-12
```

3. **Установите зависимости**

```
npm install
```

4. **Запустите проект**

```
npm run dev
```
